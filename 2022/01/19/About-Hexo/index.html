使用Hexo在github上搭建博客网站

  • Hexo
  • 前端开发
  • 静态页面

前提条件

  • Node.js (Node.js 版本需不低于 10.13，建议使用 Node.js 12.0 及以上版本)
  • git

使用npm安装Hexo

使用下列指令建一个文件夹

配置_config.yml

这步操作可以后面做，里面都是有默认值，且简单易懂，具体参考Hexo文档配置页面

如何写文章

首先使用指令创建一个新文章，创建的所有文章都在source下的_posts文件夹里，如果有写过掘金一类文章的可以试着写，格式都是一样的

换主题

  • Hexo 里有许多好看的主题，我个人喜欢简单一点的，以我这个 light 主题为例

  • 展开 themes 文件夹就可以看到个 light 文件夹，接着在 _config.yml 里找到 theme

    这样主题就大功告成啦！！！

部署到github

  • 先来到 _config.yml，去到底部找到deploy

  • 然后执行三部曲

    此时我们去到 https://<添加代码的仓库名>
